# yze-go-errlast

A [`yze`](https://github.com/gomatic/yze) analyzer (group `go`, category `errors`) enforcing the gomatic Go idiom that `error` is always the **last** return value.

- **Rule:** `yze/go/errlast`
- **Binary:** `cmd/yze-go-errlast` runs it standalone.

Built on the [`go-yze`](https://github.com/gomatic/go-yze) framework.
